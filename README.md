🚢 Titanic Exploratory Data Analysis (EDA)

🌊 Project Overview

A quick but deep dive into the 1912 Titanic disaster dataset to uncover exactly what factors influenced passenger survival. This project walks through data cleaning and visual storytelling.

🛠️ Analysis Workflow

🧹 Data Cleaning: Fixed mislabeled columns, dropped useless zero-value placeholders, and cleaned up missing data.

📊 Univariate Analysis: Explored the baseline distribution of passengers by age, gender, ticket class, and fare.

📈 Bivariate Analysis: Compared individual features directly against survival rates using bar charts and histograms.

🧩 Multivariate Analysis: Combined multiple variables to see how socioeconomic status and gender interacted to affect survival.

💡 Key Insights

👩‍👧 Women & Children First: The data proves the Birkenhead drill was enforced. Females and children had drastically higher survival odds.

💰 Wealth = Survival: Passenger class was a massive barrier. 1st-class passengers survived the most, while 3rd-class passengers survived the least.

⚠️ Intersecting Factors: Socioeconomic status impacted the "women first" rule. A 3rd-class female had noticeably lower survival odds compared to a 1st-class female.
